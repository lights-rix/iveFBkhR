# 项目简介

## 核心功能
- 前后端代码
- SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包

## 适用场景
- 管理员角色：用于管理商城整体运营，包括商品管理、订单处理等。
- 商家角色：用于商品发布、订单管理、库存管理等。
- 用户角色：用于浏览商品、下单购买、评价等。

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
